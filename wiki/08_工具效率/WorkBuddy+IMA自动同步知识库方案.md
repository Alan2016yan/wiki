## 核心定义

通过 AI 助手 **WorkBuddy** 与知识库工具 **IMA** 的联动，实现资料「一句话自动同步」，大幅提升公益组织及个人的知识构建与管理效率。

**解决什么问题**：资料手动整理归档耗时、知识碎片化难以复用。

---

## 核心内容

### 一、工具组合与配置前提

| 项目 | 内容 |
|------|------|
| 工具组合 | WorkBuddy（AI助手）+ IMA（知识库） |
| 核心功能 | 自动同步链接、本地文件、文件夹监控；自然语言提问与内容生成 |
| 操作前提 | 需在 IMA 官网获取 API Key 与 Client ID，并在 WorkBuddy 中完成技能安装与密钥配置 |

### 二、三大应用场景

#### 场景一：自动同步网页/公众号文章
- 自动同步网页链接或公众号文章至指定知识库
- 支持定时抓取与内容提炼

#### 场景二：上传本地文件
- 上传本地文件（如会议纪要、PDF研报）到 IMA 知识库

#### 场景三：文件夹监控
- 监控本地文件夹，实现新文件自动同步

### 三、配置步骤

1. 在 IMA 官网获取 API 密钥
2. 在 WorkBuddy 中安装对应技能
3. 完成密钥配置以建立连接
4. 开始使用自然语言指令管理知识库

---

## 适用场景

- ✅ 公益机构项目资料管理
- ✅ 行业动态跟踪
- ✅ 会议纪要归档
- ✅ 政策文件整理
- ⚠️ 需要 IMA API Key，不适合纯离线用户

---

## 注意事项

- IMA API Key 需要在官网申请，有一定门槛
- 定时抓取依赖网络稳定性
- 同步后的内容仍需人工审核质量
