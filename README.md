## 📁 Structure

```

├── raw/           # Original sources (read-only)
├── wiki/          # Compiled knowledge (AI-generated)
```

## 🛠️ Workflow

1. Add content to `raw/` or `wiki/`
2. Push to `main` branch
3. GitHub Actions automatically deploys to GitHub Pages
